## 一、市场定位与竞品分析

### 1.1 对标：KK 官方对战平台
- **用户规模**：百万级，国内最大 War3 RPG 对战平台
- **商业模式**：免费畅玩 + 战令 + 地图内购 + 创作者分成
- **核心壁垒**：UGC 生态闭环 — 头部地图作者月入超百万，形成正循环
- **技术栈**：Next.js 前端 + 网易 CDN + War3 引擎

### 1.2 OpenForge 差异化定位
> "下一代 RPG 地图创作平台——脱离 War3 引擎限制，面向全球市场"

| 维度 | KK 对战平台 | OpenForge |
|------|-----------|-----------|
| 引擎 | War3（20年老引擎） | Godot 4.6（现代开源） |
| 画面 | 受限于 War3 | 无限制（2D/3D） |
| 市场 | 中国大陆 | **全球**（多语言 I18n） |
| 分发 | 自有平台 | **Steam + 自有平台** |
| 移动端 | 有限支持 | Godot 原生多端 |
| 创作门槛 | W3 编辑器（高） | JSON 数据驱动（低） |

## 二、旗舰地图：不思议求生（Rogue Survivor）

### 2.1 核心玩法公式
**Roguelike 随机性 × 卡牌构筑 × 塔防策略 × RPG 成长 = 无限可重玩性**

参考不思议作战的成功要素：
- 每局随机卡组 → 每局体验不同
- 羁绊/套装系统 → 策略构筑深度
- 转职系统 → 关键战略抉择
- 难度分级 → 持续挑战目标

### 2.2 升级路线设计

```
Lv.1 ──── Lv.5（转职）──── Lv.10（二转？）──── Lv.20（觉醒？）
  │           │                  │                    │
  │    选择进阶职业         选择觉醒路线           终极形态
  │   （2选1）             （3选1？）             （独特大招）
  ▼           ▼                  ▼                    ▼
每级3选1卡片  解锁新被动     解锁终极技能         专属套装加成
```

### 2.3 数据成长曲线

| 等级 | 基础 DMG | 攻速 | 生命 | 波次 | 敌人血量 |
|------|---------|------|------|------|---------|
| 1 | 12-35 | 1.0-1.4s | 120-250 | 1-5 | 40-80 |
| 5 | 50-100 | 0.6-1.0s | 300-500 | 10-15 | 100-200 |
| 10 | 100-250 | 0.4-0.7s | 500-1000 | 20-30 | 200-500 |
| 15 | 250-500 | 0.3-0.5s | 800-1500 | 40+ | 500-1000 |
| 20 | 500+ | 0.2-0.4s | 1500+ | 60+ | 1000+ |

**设计原则**：
- 前 5 级快速成长（爽感期）
- 5-10 级套装成型期（策略感）
- 10-15 级 BOSS 挑战期（紧张感）
- 15+ 级爬塔无尽模式（自我超越）

## 三、局内系统设计

### 3.1 波次系统（已实现 + 扩展）
```
普通波 → 每7秒
精英波 → 每5波出1个精英怪（金色边框，3倍血量，掉落保底蓝色卡）
BOSS 波 → 每2分钟
终极 BOSS → 10分钟
无尽模式 → 击杀终极BOSS后进入，难度指数增长
```

### 3.2 宝物系统（新增，参考不思议作战）
- 每击杀 50 个敌人解锁一次"宝物选择"（3选1）
- 宝物提供独特被动效果（不是卡片，是额外层级）
- 宝物示例：
  - 🔮 **先知之眼**：看到下 3 波的怪物组成
  - ⚡ **雷神之锤**：每 30 秒自动释放一次全屏闪电
  - 🛡️ **不屈意志**：血量低于 10% 时免疫 3 秒（60s CD）
  - 🎲 **赌徒之心**：所有概率效果 ×2，但受到伤害 ×1.5

### 3.3 精英词条系统（新增）
- 精英怪携带随机词条（类似暗黑破坏神）：
  - 反射：反弹 20% 伤害
  - 闪电护盾：近距离攻击者受电击
  - 分裂：死亡时分裂为 2 个小怪
  - 吸血：攻击回血
  - 冰冻光环：减速附近玩家

- **赛季周期**：30 天
- **免费通道**：基础奖励（金币/Star Dust/基础皮肤）

  - 赛季限定卡背
  - 额外 Star Dust ×3
  - 赛季专属宝物

### 4.3 成就系统
- **局内成就**：单局击杀 500+、10 分钟内通关、全套装集齐
- **累计成就**：总击杀 10000、游玩 100 局、集齐所有套装
- **隐藏成就**：特殊操作触发（如 0 卡通关、1 血通关）
- **成就奖励**：Star Dust + 专属称号 + 解锁隐藏职业

### 4.4 收藏系统
- **图鉴**：解锁所有卡片/套装/宝物/BOSS
- **完成度奖励**：每 25% 解锁一个永久加成
- **皮肤收藏**：英雄皮肤/弹道皮肤/技能特效

## 十、核心结论

1. **不思议作战验证了 Roguelike+构筑 是最热公式**——OpenForge 的 rogue_survivor 方向正确
2. **KK 平台的 UGC 生态闭环是终极目标**——创作者能赚钱才能持续产出内容
3. **战令系统是最成熟的变现模板**——$4.99/月 的价格带在全球市场被广泛接受
4. **全球化是最大差异化优势**——KK 只做中国市场，OpenForge 面向全球
5. **先做好一张地图，再做平台**——不思议作战单图就撑起了 KK 的增长
6. **绝不卖数值**——Pay-to-Win 会毁掉游戏口碑和 Steam 评价

## 附录 C：OpenForge 与不思议作战差距分析

### C.1 功能对比

| 系统 | 不思议作战 | OpenForge | 差距 |
|------|-----------|-----------|------|
| 卡组+羁绊 | 大量+隐藏跨组羁绊 | 38卡+14套装 | **需加跨组主题羁绊** |
| 转职 | 保留前英雄技能叠加 | 一次性选择 | **需加传承机制** |
| 宝物系统 | 3选1独立被动 | 无 | **缺失** |
| 小世界 | 独立BOSS副本 | 无 | **缺失** |
| 词条锻造 | 玩家DIY词条方向 | ItemSystem有装备无锻造 | **需加锻造** |
| 局外养成 | 6条成长线 | 2条（天赋+许愿） | **需扩到5+** |
| 累抽保底 | 扭蛋+许愿+宝库三层 | 单层许愿池 | **需加累抽** |
| 赛季 | 有 | 无 | **缺失** |

### C.2 OpenForge 优势
1. **技术架构**：SpellSystem/AuraManager/ProcManager >> War3 JASS
2. **跨平台**：Godot → PC/移动/Web
3. **国际化**：4语言 >> KK 仅中文
4. **数据驱动**：新内容只需 JSON >> War3 需重新编译
5. **框架通用**：同框架可做多种 GamePack

### C.3 高优先级补齐项

| 优先级 | 系统 | 预估工作量 | 影响 |
|--------|------|-----------|------|
| P0 | 宝物系统（3选1被动） | 2天 | 每局策略深度+1层 |
| P0 | 跨组主题羁绊 | 3天 | 构筑多样性×3 |
| P1 | 小世界副本（分支场景） | 3天 | 节奏体验质变 |
