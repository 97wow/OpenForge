## 一、核心公式

### 1.1 伤害公式
```
最终伤害 = (基础DMG + 主属性加成) × (1 + 百分比加成) × (1 + 暴击期望)
```

### 1.2 有效生命
```
EHP = MaxHP / (1 - 护甲减伤率)
护甲减伤率 = armor / (armor + 100)  # 100护甲=50%减伤
```

### 1.3 DPS
```
DPS = 最终伤害 / 攻速间隔 × (1 + 暴击率 × (暴击倍率-1))
```

- 每波怪物总 EHP ≈ 玩家当前 DPS × 7秒（1 波间隔）× 1.2（留余量）
- 怪物 DPS ≈ 玩家 EHP / 60秒 × 难度系数
